⚡ Enterprise Deployment ✕
One-click Kubernetes + WebRTC deployment. Generates all manifests, configures mesh networking, and provisions governance policies.
Deployment Progress
1
Namespace + RBAC
Creates causalmesh namespace with zero-trust RBAC policies and network policies.
kubectl create namespace causalmesh kubectl apply -f rbac-mesh.yaml kubectl apply -f networkpolicy-zerotrust.yaml
2
Shard StatefulSets (6x replicas)
Deploys auto-sharded ledger nodes with anti-affinity rules across AZs.
kubectl apply -f shard-statefulset.yaml kubectl rollout status statefulset/causalmesh-shard -n causalmesh
3
Inference Router + WebRTC Mesh
Deploys multi-model inference router with WebRTC signaling server and STUN/TURN.
kubectl apply -f inference-deployment.yaml kubectl apply -f webrtc-signaling-svc.yaml kubectl apply -f stun-turn-configmap.yaml
4
Clock Synchronization Service
Initializes global Lamport + per-node vector clock daemons with cryptographic ordering.
kubectl apply -f lamport-daemon-ds.yaml kubectl apply -f vector-clock-configmap.yaml kubectl apply -f clock-sync-cronjob.yaml
5
Governance + Audit CRDs
Installs Custom Resource Definitions for policies, audit proofs, and rollback triggers.
kubectl apply -f governance-crd.yaml kubectl apply -f audit-policy-crd.yaml kubectl apply -f rollback-trigger-crd.yaml
6
Monetization + Metering Layer
Deploys token accounting, usage metering, and enterprise billing webhook.
kubectl apply -f metering-sidecar-inject.yaml kubectl apply -f billing-webhook-deployment.yaml # Mesh ready. Endpoint: https://mesh.causalmesh.svc:8443
